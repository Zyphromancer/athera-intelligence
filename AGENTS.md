# Agent Notes

- This project is a TanStack Start application with file-based routing in `src/routes/`.
- Do not create `src/pages/` or Next.js-style layouts; the root layout is `src/routes/__root.tsx`.
- `src/routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin.
- Server-only code lives in `*.server.ts` modules or TanStack server functions (`*.functions.ts`).
- Avoid importing server-only modules into client components.
- Keep design tokens in `src/styles.css`; avoid hardcoded color utilities in components.
- Run `npm run build` after structural changes to regenerate the route tree and catch TypeScript errors.
